# Operazioni sulle righe

Nella **modalità testo** (l'editor sorgente) puoi manipolare righe intere con scorciatoie:

- **Sposta riga su/giù**: `Ctrl+Shift+↑` / `Ctrl+Shift+↓`
- **Duplica riga**: `Ctrl+D`
- **Elimina riga**: `Ctrl+L`

## Perché solo in modalità testo

In modalità WYSIWYG (formattata) il concetto di "riga" non è netto: il documento è fatto di
blocchi (paragrafi, titoli, liste). Per questo le operazioni-riga sono **mode-aware**: attive
nel testo, mentre in WYSIWYG quegli stessi tasti mantengono la loro funzione di formattazione.
