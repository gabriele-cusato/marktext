# Editing del testo

Oltre a scrivere, MarkText offre comodità per lavorare sul testo più in fretta.

## Cosa puoi fare

- [**Ingrandire/rimpicciolire il testo**](lvl2-zoom/index.md){ .lvl-link .lvl-2 } tenendo premuto **Ctrl** e
  girando la rotella del mouse. Cambia solo la dimensione del testo, non di tutta l'interfaccia.
- **Mandare a capo le righe lunghe** (word wrap) nella modalità testo, così non devi
  scorrere in orizzontale.
- **MAIUSCOLO / minuscolo**: trasformi la selezione con una scorciatoia.
- [**Operazioni sulle righe**](lvl2-operazioni-riga/index.md){ .lvl-link .lvl-2 } (nella modalità testo):
  spostare, duplicare o eliminare una riga al volo.
