# Salvare i file

Quando scrivi in MarkText, il tuo testo vive in una scheda. **Salvare** significa
scrivere quel testo sul file vero e proprio nel tuo computer.

## Cosa ti offre

- **Sai sempre se hai modifiche non salvate.** Sulla scheda compare un piccolo
  [**pallino "non salvato"**](lvl2-bollino/index.md){ .lvl-link .lvl-2 }: c'è = hai cambiato qualcosa che non è
  ancora sul disco; sparisce appena salvi.
- **Salvi con `Ctrl+S`.** Veloce e immediato; quello che vedi è esattamente quello che
  finisce nel file.
- **MarkText rispetta il tuo file.** Se apri un documento creato da un altro programma,
  quando salvi mantiene il suo [**formato originale**](lvl2-formato-eol/index.md){ .lvl-link .lvl-2 } (codifica
  dei caratteri e tipo di "a capo"), senza stravolgerlo.
