# Finestre e schede

Ogni documento aperto vive in una **scheda** (tab) nella barra in alto, come nei browser.

## Cosa puoi fare

- **Aprire più file insieme**: ognuno è una scheda separata, passi dall'una all'altra con un clic.
- [**Riordinare le schede**](lvl2-gestione-tab/index.md){ .lvl-link .lvl-2 }: trascinale a destra/sinistra per
  metterle nell'ordine che vuoi.
- **Aprirne di nuove**: con il pulsante **+**.
- **Chiuderle**: con la **×** sulla scheda.
- [**Spostare la finestra**](lvl2-drag-finestra/index.md){ .lvl-link .lvl-2 }: trascina uno spazio vuoto della
  barra delle schede, come una normale barra del titolo (doppio clic = ingrandisci/ripristina).
